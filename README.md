Testautomation Projekt
        Beskrivning

Detta projekt är en enkel testautomationslösning utvecklad i Python med Pytest. Syftet är att visa grundläggande färdigheter inom testautomation och testramverk, inklusive filvalidering, API/databas-test och integration med CI/CD-pipeline.

        Struktur
- Tre olika tester:
    1. Enkelt test som alltid passerar
    2. Filinnehållstest
    3. API- eller databas-test

- Tester kan köras lokalt eller i CI/CD-pipeline (GitHub Actions/BitBucket Pipelines)

        AI-användning
- Jag använde AI för att förstå bibliotek som Pandas och SQLAlchemy, samt för debugging och felsökning när jag stötte på problem.
- All testlogik, designval och pipeline-konfigurationer har implementerats och verifierats av mig själv.

        Rapportering
- Testresultaten kan visualiseras i terminalen eller med verktyg som Allure Test Report för bättre presentation.
